### How to Run the Benchmark
1. Place the benchmark program in the root of this repository.
2. Compile it with:
   ```bash
   g++ -std=c++17 benchmark.cpp -o benchmark
   ```
3. Run the benchmark:
   ```bash
   ./benchmark
   ```
